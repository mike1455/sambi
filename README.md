# Сайт САМБи — Столичная академия малого бизнеса

Современный сайт института с админ-панелью и формой обратной связи.
Технологии: Next.js 15 + React 19 + TypeScript, без базы данных и внешних сервисов —
контент хранится в JSON-файлах на сервере.

## Что внутри

- **Публичный сайт**: главная, «Об академии», направления обучения (4 программы),
  «Абитуриентам» (шаги, документы, стоимость), новости, отзывы, контакты,
  «Сведения об образовательной организации», политика обработки персональных данных.
- **Форма обратной связи** — на главной и на странице «Контакты»: с согласием на
  обработку персональных данных (152-ФЗ), защитой от спама (honeypot + ограничение
  частоты). Заявки попадают в админку.
- **Админка** — `/admin`: заявки с сайта, редактирование всех текстов, новостей,
  отзывов, FAQ, направлений, контактов и настроек. Изменения видны на сайте сразу.

## Запуск локально

```bash
npm install
npm run dev
```

Сайт: http://localhost:3000 · Админка: http://localhost:3000/admin

## Пароль админки

Стартовый пароль — `sambi2026`. **Обязательно смените его** — задайте переменную
окружения перед запуском:

```bash
ADMIN_PASSWORD="ваш-надёжный-пароль" npm start
```

или создайте файл `.env.local`:

```
ADMIN_PASSWORD=ваш-надёжный-пароль
```

Пока используется стартовый пароль, админка показывает предупреждение.

## Публикация (продакшн)

Контент и заявки записываются в файлы `content/data/*.json`, поэтому сайту нужен
хостинг с постоянным диском (обычный VPS или Node-хостинг: Timeweb, reg.ru,
Beget и т. п.):

```bash
npm install
npm run build
ADMIN_PASSWORD="ваш-пароль" npm start   # порт 3000
```

Дальше — любой reverse-proxy (nginx/caddy) с HTTPS на домен `sambimsk.ru`.

> **Vercel/Netlify**: сайт и форма открываться будут, но сохранение контента и заявок
> не сохранится между перезапусками — файловая система там только для чтения.
> Для полноценной работы админки используйте сервер с диском.

## Резервные копии

Весь редактируемый контент лежит в `content/data/` (создаётся при первом сохранении).
Достаточно периодически копировать эту папку.

## Структура

```
app/            — страницы (App Router) и API
  admin/        — админ-панель
  api/          — форма обратной связи + API админки
components/     — шапка, подвал, форма, иконки
lib/            — контент (JSON-хранилище), авторизация
content/data/   — сохранённый контент и заявки (в git не попадает)
public/         — логотип, шрифты, robots.txt
```
